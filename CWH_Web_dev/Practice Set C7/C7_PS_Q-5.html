which of the following is used to look for the ancestor that matches a given CSS Selector Options are :- 1. matches 2.closest 3.contains 4. none of these Answer - closest() discription - The closest() method is used to look for the ancestor that matches a given CSS selector. It traverses up the DOM tree, starting from the current element, and returns the first ancestor element that matches the specified selector. If no matching ancestor is found, it returns null.
